# stoch3com

Crypto wheeling-dealing

## Usage

1. Docker via docker compose:
```bash
$ docker compose build
$ docker compose up -d
```

   * Diving into:
   ```bash
   $ docker exec -it $(docker compose ps -q stoch3com-bot) bash
   ```

2. Poetry:
```bash
$ poetry install
$ poetry run bot
```
